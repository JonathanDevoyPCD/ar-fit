AR-FIT Front-End Files

Files included:
- index.html
- workout.html
- meals.html
- assets/css/styles.css
- assets/scss/styles.scss

How to use:
1. Keep the folder structure intact.
2. Open index.html in your browser.
3. Edit the meal and workout content directly in workout.html and meals.html.
4. If you want to continue building this into a full app, the current structure is ready for:
   - login/auth screens
   - check-off progress states
   - workout history
   - meal tracking
   - hydration logging
   - mobile nav
